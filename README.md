# Vintage Verse - E-Commerce Platform

A modern vintage fashion e-commerce website with an integrated admin panel for complete business management.

## 🎯 Features

### Public Website
- Beautiful responsive design
- Featured product collection
- Shopping cart functionality
- Smooth navigation and animations
- Mobile-friendly layout

### Admin Panel
- **Dashboard** - Overview of sales, products, orders, and customers
- **Products Management** - Add, edit, and delete products
- **Orders Management** - View and update order status
- **Customers** - View customer information and purchase history
- **Coupons** - Create and manage discount coupons
- **Settings** - Configure site settings

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/rafiushariar-coder/VINTAGE-VERSE.git
   cd VINTAGE-VERSE
   ```

2. **Open in browser**
   - Open `index.html` in your browser to view the public website
   - Open `admin.html` in your browser to access the admin panel

3. **Local Storage**
   - All data is stored in browser's localStorage
   - No backend required for basic functionality

## 💻 Usage

### For Customers
1. Browse the featured collection
2. Click "Add to Cart" to add items
3. Navigate through different sections

### For Admin
1. Go to `admin.html`
2. Use the sidebar navigation to access different sections
3. Manage products, orders, customers, and coupons
4. View dashboard statistics

## 🎨 Customization

### Change Colors
Edit the color values in the CSS sections:
- Primary color: `#b08d57` (gold)
- Dark color: `#2d241c` (dark brown)
- Background: `#f5efe6` (beige)

### Add Products
In admin panel, click "Add New Product" and fill in the details.

### Create Coupons
In admin panel, go to Coupons section and create new discount codes.
